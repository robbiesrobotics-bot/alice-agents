## TL;DR

Every major lab now ships "agents" as a first-class product — but most are **single-model, platform-locked**, and priced for enterprise from day one. A.L.I.C.E.'s model-agnostic, indie-first positioning + new Enterprise tier is well-timed, but the window to own "team of specialists you install in 60 seconds" is closing. ## 1. What's New (Last 7 Days, March 2026)

**OpenAI**
- Operator/computer-use capabilities maturing. Tight ChatGPT integration but still not a dev framework for building multi-agent teams — not a direct threat.
- GPT-5 likely landed or is imminent. Expect "one powerful model is enough" counter-narrative to specialist teams. A.L.I.C.E. needs a clear rebuttal ready.

**Anthropic**
- MCP (Model Context Protocol) adoption spreading. Claude native agent orchestration improving but still requires dev assembly. A.L.I.C.E.'s NemoClaw compatibility + MCP awareness is a hook.
- Claude Teams tier eating enterprise budget — not a framework competitor, but worth noting.

**Google**
- Vertex Agent Builder is capable for enterprise workflows but cloud-native, Python/Java-centric, per-call pricing. Not indie-friendly. A.L.I.C.E. differentiates cleanly here.

**Startups & OSS**
- **CrewAI**: Still A.L.I.C.E.'s closest conceptual competitor (role-based agents, task delegation). Raising money and shipping fast. Watch pricing moves — this is the one to track.
- **Agno (ex-Phidata)**: Rebranded, gaining GitHub/X mindshare fast with the "build your own agent team" crowd. Credible threat to A.L.I.C.E.'s developer audience.
- **LangGraph**: Go-to for stateful multi-agent graphs among developers. Code-first, low-level — A.L.I.C.E. is high-level. Ensure the power-user upgrade path from A.L.I.C.E. → LangGraph is a complement, not a cliff.
- **Dify / Flowise / n8n**: No-code/low-code agent builders eating the non-developer workflow market. Mission Control's visual editor competes here.
