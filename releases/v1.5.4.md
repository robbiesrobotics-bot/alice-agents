# v1.5.4

Status: approved

## Summary

Rewrites the cloud onboarding flow to use Supabase OAuth via browser instead of manual ingest tokens. Zero-friction setup for non-technical users.

## Highlights

- Browser-based OAuth login replaces manual token copy-paste
- Auto-detect gateway URL/token from openclaw.json
- Auto-register gateway with alice.av3.ai
- Heartbeat daemon setup during install
- Upgrade mode shows existing cloud status

## Announcement

A.L.I.C.E. v1.5.4 makes cloud setup frictionless. Pro users who enable Cloud get browser-based OAuth — no tokens to copy-paste, no config files to edit. The installer auto-detects your gateway and registers it.
