# v1.5.10

Status: approved

## Summary

Hermes-first install path — A.L.I.C.E. now runs natively on Hermes Agent as a skill tree, with automatic runtime detection and zero model-selection wizard.

## Changes

- **Hermes Agent install path**: New `lib/hermes-installer.mjs` — full install, uninstall, and status for Hermes runtime
- **Hermes skill templates**: `hermes-orchestrator-skill.md` + `hermes-specialist-skill.md` for A.L.I.C.E. orchestrator and all 33 specialists
- **Runtime auto-detection**: `detectRuntime()` checks Hermes first, then NemoClaw, then OpenClaw
- **`--runtime` flag**: Force a specific runtime (`--runtime hermes|openclaw|nemoclaw`)
- **No model wizard**: Removed all model selection prompts — uses whatever the runtime already has configured
- **Fail-fast on missing model**: Exits with clear instructions if no model is detected
- **README rewrite**: Hermes as recommended runtime, updated docs, no model selection section

## Announcement
