# v1.4.5

Status: approved

## Summary
- Mission Control Cloud nodes now send heartbeats to the hosted runtime and `admin.av3.ai`
- The bundled bridge now supports machine-to-admin heartbeat delivery using the shared worker token
- Cloud installs record the admin heartbeat endpoint so the setup stays portable across machines

## Announcement
`v1.4.5` makes the Mission Control Cloud add-on operationally visible. Hosted nodes now report heartbeats to both the runtime and the admin portal, which means Fleet and Incidents can reflect real node health instead of static install state.
