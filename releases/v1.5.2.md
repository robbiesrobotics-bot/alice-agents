# v1.5.2

Status: approved

## Summary
Agent roster expanded from 28 to 31 specialists. Two new Pro-tier agents added: Aria (Autonomous Research Agent) and Nate (n8n Workflow Automation Specialist). Landing page, README, and all documentation updated to reflect the full roster.

## What's New

**Aria — Autonomous Research Agent (Pro)**
Aria is a long-running autonomous research agent that lives per project. She spins up sub-agents for parallel research, sets up cron jobs to improve low-scored topics every 4 hours, and writes findings to a file-based queue on Ubuntu Desktop. Currently active on: Namibia AI Infrastructure project and OperationBLLM (community-owned AAVE LLM).

**Nate — n8n Workflow Automation Specialist (Pro)**
Nate owns the automation layer for the full A.L.I.C.E. system. He designs, builds, deploys, and maintains n8n workflows exclusively via the n8n REST API — for reproducibility and version control. Nate is the bridge between all A.L.I.C.E. components, integrating Supabase, Telegram, Stripe, GitHub, and the broader automation stack.
