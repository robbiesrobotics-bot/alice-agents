# v1.4.6

Status: approved

## Summary
- Added a provider-aware bundled `coding-agent` skill so coding workflows match the user's OpenClaw default model automatically
- Fixed npm packaging so Starter and Pro rosters are both shipped correctly and Pro installs are assembled from the starter set plus Pro add-ons
- Hardened install state, license validation, and `--doctor` reporting so onboarding is more predictable and easier to support

## Announcement
`v1.4.6` is a production-hardening release for the A.L.I.C.E. installer. New users now get a coding workflow that follows their OpenClaw default provider automatically, the published package ships the real Starter and Pro rosters, and Pro licensing plus doctor diagnostics are much clearer when something needs attention.
