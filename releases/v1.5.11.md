# v1.5.11

Status: approved

## Summary

Runtime auto-install wizard — when no runtime is detected, A.L.I.C.E. now offers to install Hermes Agent, OpenClaw, or NemoClaw automatically, with verified install commands and post-install guidance.

## Changes

- **New: `lib/runtime-installer.mjs`** — full runtime auto-install module with wizard UI, install commands for all 3 runtimes, pre-flight checks (Docker for NemoClaw, macOS warnings), post-install steps
- **Updated `detectRuntime()`** — now returns `null` when nothing found (instead of silently falling back to OpenClaw), checks NemoClaw binary first
- **Updated `runInstall()`** — handles 3 new cases: no runtime detected, `--runtime X` but X not installed, and normal flow unchanged
- **NemoClaw first-class support** — proper detection, install via `curl -fsSL https://www.nvidia.com/nemoclaw.sh | bash`, Docker pre-flight check

## Announcement

A.L.I.C.E. v1.5.11 adds a runtime auto-install wizard. New users running `npx @robbiesrobotics/alice-agents` without any runtime installed now get a guided setup: choose Hermes Agent (recommended, ~2 min), OpenClaw (~3 min), or NemoClaw (~20-30 min, sandboxed). The installer runs the official install command, verifies it worked, and tells you exactly what to do next. NemoClaw is now first-class alongside Hermes and OpenClaw.
