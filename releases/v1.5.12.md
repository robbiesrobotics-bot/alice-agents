# v1.5.12

Status: approved

## Summary
- Unified the 33-agent roster across Hermes Agent and OpenClaw/NemoClaw installs so the published product matches the documented team
- Hardened runtime state, reinstall behavior, and Hermes cleanup so upgrades and cross-runtime installs are more predictable
- Sanitized the published npm artifact to ship only generic shared workspace templates, removing local machine and user-history data from the public package

## Announcement

A.L.I.C.E. v1.5.12 is a release-hardening update focused on consistency and privacy. All supported runtimes now install the same 33-agent team, runtime selection is more stable across upgrades, and the public npm package has been trimmed down to generic shared templates so local user history and machine-specific workspace content no longer ship in the artifact.
