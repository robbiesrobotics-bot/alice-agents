# v1.5.1

Status: approved

## Summary
Token provisioning system for Mission Control multi-tenant authentication. Pro purchases now auto-generate ingest + worker tokens at checkout, local bridge events are verified flowing live to alice.av3.ai, and the admin portal gains inline org editing.

## Announcement
`v1.5.1` closes the loop between a Pro purchase and a live Mission Control connection. When you buy Pro, Stripe now provisions two tokens alongside your license key — an ingest token and a worker token — and emails all three together. The installer accepts these tokens and writes them to your local bridge config so events start flowing to `alice.av3.ai` immediately. Each install's data is isolated by `team_id` in Supabase, so `alice.av3.ai` works as a true multi-tenant hosted dashboard. This release also ships live end-to-end verification (ingest accepted, node registered) and wires the admin portal's Organizations view to real Supabase data with inline editing.
