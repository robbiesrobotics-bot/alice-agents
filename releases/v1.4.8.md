# v1.4.8

Status: approved

## Summary
- Pro license validation is now fully operational — the production landing site was missing Supabase credentials and the licenses table had never been created
- Stripe production key corrected from test to live mode
- Pricing inconsistencies fixed across schema.org markup and transactional emails
- Documentation expanded with 7 new pages covering FAQ, licensing, troubleshooting, configuration, and per-tier agent rosters

## Announcement
`v1.4.8` fixes the Pro license activation flow end-to-end. License keys are now properly created on checkout, stored in Supabase, and validated when users run `npx @robbiesrobotics/alice-agents --license <key>`. If you previously couldn't activate a Pro license, re-run the installer with your key — it will work now.
