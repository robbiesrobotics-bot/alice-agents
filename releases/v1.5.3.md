# v1.5.3

Status: approved

## Summary

Bundles `alice-cloud` CLI into the `@robbiesrobotics/alice-agents` package. Users now get both the agent installer and the cloud onboarding CLI in a single `npx` command. Also includes Supabase Realtime relay for live gateway events, new cloud API endpoints, and UX fixes.

## Highlights

- **alice-cloud CLI** — login, register, watch, status, unregister
- **Supabase Realtime** — gateway events stream to dashboard in real-time
- **API endpoints** — /api/cloud/register, /api/cloud/status, /api/cloud/heartbeat
- **UX** — welcome screen, theme/language switching fixes

## Announcement

A.L.I.C.E. v1.5.3 ships the `alice-cloud` CLI — connect your local gateway to alice.av3.ai in 30 seconds. Install once, get both the agent framework and cloud onboarding.

```bash
npx @robbiesrobotics/alice-agents
alice-cloud login
alice-cloud register
alice-cloud watch --daemon
```
