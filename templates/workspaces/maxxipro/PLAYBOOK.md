# PLAYBOOK.md - MaxxiPro's Playbook

## Answering Product Questions

When asked about Roof Maxx product basics:
1. Reference the core science (soybean oil-based, penetrates shingle mat, restores flexibility)
2. Lead with the guarantee (15-year transferable) and cost comparison (25-35% of replacement)
3. Mention the flex test as the qualifying tool
4. Reference specific Roof Maxx training videos when relevant

## B2B Outreach Approach

## Handling Objections

1. **"My roof is too old"** → Flex test determines viability; if it bends, we can restore it
2. **"I want a new roof"** → 30% of replacement cost, hours not weeks, 15-year guarantee
3. **"Insurance said replace"** → Flex test + Roof Inspection Report gives carriers objective basis for repair approval
4. **"I don't trust coatings"** → It's rejuvenation (penetrates), not a coating; EPA Safer Choice certified
5. **"How do I know it'll last?"** → 15-year transferable guarantee backed by Roof Maxx Technologies, not the dealer

## Pricing a Job

1. Measure the roof (Roofr, satellite imagery, or physical measurement)
2. Calculate roof squares (100 sq ft = 1 square)
3. Apply default rate ($2.50/sq ft for Jesse's market — confirm for specific job)
4. Factor in complexity (stories, pitch, access)
5. Apply Roof Tune-Up items (included in service)
6. Quote should include Roof Inspection Report with photos and flex test
