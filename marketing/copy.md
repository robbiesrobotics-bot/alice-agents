### FAQ

**Q: Does this work with my existing OpenClaw setup?**
Yes. A.L.I.C.E. agents install on top of OpenClaw. If you're already running it, you're halfway there.

**Q: Which AI models do the agents use?**
That's up to you. A.L.I.C.E. works with Anthropic (Claude), OpenAI (GPT), and local models via Ollama. Each agent uses whichever model you configure in OpenClaw.

**Q: What happens to my data?**
On Starter and self-hosted Pro, everything runs locally. Cloud sync (Pro) is opt-in and stores memory state only — not your full conversation history. MIT licensed, so you can verify exactly what's happening.

**Q: Can I build on top of this or modify the agents?**
Absolutely. MIT license means you can fork it, customize agent personas, add your own specialists, or integrate it into something else entirely. We'd love to see what you build.

### Key Differentiators vs. Tools That Hardcode Claude/OpenAI

- **No forced provider** — Most AI agent frameworks require you to sign up for Anthropic or OpenAI before you see a single result. A.L.I.C.E. auto-detects whatever model you have configured in OpenClaw/NemoClaw and works immediately.
- **No wasted API keys** — If you're already on OpenClaw with a model configured, you're ready. No second account, no second billing relationship.
- **Works with local models** — Running Ollama? That works too. A.L.I.C.E. isn't in the business of pushing you toward cloud spend.
- **Truly zero-config start** — The "no account needed" claim is real. `npx @robbiesrobotics/alice-agents` works without any API setup if you have OpenClaw running.
- **Future-proof** — When better models ship, you switch once in OpenClaw. All 28 agents follow. No individual agent configs to update.
